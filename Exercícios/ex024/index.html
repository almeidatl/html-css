Neste exercício, vamos explorar o uso do atributo target="name" em iframes. O atributo target especifica em qual janela ou quadro um link deve ser aberto quando clicado. No caso de iframes, podemos usá-lo para direcionar o link que escolher abaixo para que, em vez de abrir uma nova página, a mesma seja apresentada nesse quadradinho ao lado!
